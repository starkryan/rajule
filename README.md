# Rohit Rayaan's Portfolio

Welcome to the code repository for **Rohit Rayaan's Portfolio**. This is a responsive, animated portfolio web application built with **Next.js** and **React**, designed to showcase Rohit's work, experience, and skills.

## Features

- **Animated Introduction**: A typewriter animation introduces Rohit as a mechanical engineer, computer science enthusiast, and web developer.
- **Interactive Like Button**: Users can "like" the portfolio by clicking the heart icon, toggling its color between red and gray.
- **Responsive Design**: Mobile-first design using Tailwind CSS.
- **Dynamic Content**: Uses Framer Motion for animations and transitions.
- **Contact Section**: Includes a contact button to email Rohit directly.
- **Social Media Links**: Instagram profile link for direct connection.

## Technology Stack

- **Next.js**: For server-side rendering and routing.
- **React**: The front-end JavaScript library.
- **Tailwind CSS**: For utility-first, responsive styling.
- **Framer Motion**: For animations and transitions.
- **Lucide Icons**: Used for social icons and additional graphics.
